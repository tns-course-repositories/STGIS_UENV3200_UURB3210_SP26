Spatial Analysis & Geoprocessing - Vector Data

Preamble

Within the vector model, a core GIS capacity is the ability to identify, manipulate, transform and aggregate both within (single-layer) and between (multi-layer) feature(s).

To do this, lower level tools - Select by Expression and Select by Location - are extended by higher level tools that will be explored in this week’s demonstration lab and assignment. These tools are deployed in order to solve specific spatial problems. In this demonstration lab, the spatial problems will be rather limited to simply demonstrate how the tools work. In the assignment, relatively complex spatial problems will be solved for a real-life challenge poised by Explosive Remnants of War (ERW) based on data from the late 1990’s Kosovo War.

Demo Step 1 - Preparing a Project Environment for Geoprocessing

In order to ensure geoprocessing results in a GIS project are consistent and manageable, it is critical to ensure that both the project file and each spatial layer is consistent in the same projection system. For this demonstation lab, all data will be projected to New York’s local CRS - EPSG:2263 - NAD83/New York Long Island (ftUS).

Second, a consistent output location should be determined. This can be the temporary memory or a particular directory - for this lab we will use a folder in the data directory titled exports.

That is not the case with the higher level geoprocessing tools that we will now turn our attention towards in the next steps.

Demo Step 5 - Multiple Layer Analysis - Union

In the next geoprocessing example, we will utilize the Manhattan Buildings and the Manhattan Parcel Layer in a Union process. In this process it is often the case that we are trying to ‘merge’ attributes into one attribute table; and we don’t prioritize the resulting geometry. Unions are often used as intermediate steps in an analysis methodology, or they can be the very last stage prior to selecting by attributes some particular combination condition.

We might want to answer a simple question: according to the NYC parcel and buildings dataset, which building is the tallest, and who owns it?

With a union, we can easily answer this question in one place.

In this seventh demonstration lab, we explored both low and high level geoprocessing tools available in QGIS. These tools will be used extensively in the seventh assignment to solve an important problem - setting priority geography for a toxic or contamination.
